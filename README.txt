# 📦 Systems and Control Integration Project## 🚀 Getting Started### 1. Clone the repository```bashgit clone <repo-url>cd SystemsAndControlIntegrationProject_5SC25```### 2. Switch to your personal branch```bashgit switch your-name```---## 💻 Working on the Project### Make changes, then commit```bashgit add .git commit -m "Short description of your changes"```### Push your work```bashgit push```---## 🔀 Keeping Your Branch UpdatedBefore starting new work, always sync with `main`:```bashgit switch maingit pullgit switch your-namegit merge main```---## ⚠️ Important Rules* ❌ **Do NOT push directly to `main`*** ✅ Always work on your own branch (`thomas`, `simon`, etc.)* ✅ Keep commit messages clear and short* ✅ Pull latest changes regularly---## 🔁 Merging into `main`When your work is ready:1. Push your branch:   ```bash   git push   ```2. Create a **Pull Request** on GitHub3. Get it reviewed before merging---## 🧠 Tips* Check your current branch:  ```bash  git branch  ```* Check what changed:  ```bash  git status  ```* If something breaks: ask before forcing anything 😄---## 👥 BranchesEach team member has their own branch:* `thomas`* `simon`* `alexis`* `costin`---Happy coding 🚀
